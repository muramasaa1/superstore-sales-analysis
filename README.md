# Superstore Sales Analysis

Этот проект представляет собой комплексный анализ продаж розничной сети (датасет Sample Superstore). 
Реализован полный цикл обработки данных: от очистки в PostgreSQL до визуализации в Power BI.

## Технологический стек
* **База данных:** PostgreSQL
* **Визуализация:** Power BI
* **Языки/Инструменты:** SQL, DAX

## Основные этапы
1. **SQL:** Загрузка данных, настройка типов (исправление формата чисел с запятой на точку), создание аналитических представлений (Views).
2. **DAX:** Создание таблицы календаря (Date Table), расчет KPI (Total Sales, Total Orders, Avg Order Value, Cumulative Sales).
3. **Power BI:** Создание интерактивного дашборда с фильтрацией по регионам, годам и способам доставки.

## Инсайты
* **Сегментация:** Сегмент Consumer является основным источником дохода (51,29%), что делает B2C-маркетинг приоритетным направлением для компании.
* **Сезонность:** Анализ временных рядов показывает выраженный пик продаж в ноябре (12,3 млн), в то время как февраль является периодом минимальной активности (2,7 млн).
* **Ассортимент:** Категория Office Supplies лидирует по объему продаж, что указывает на высокую оборачиваемость товаров этого сегмента.
